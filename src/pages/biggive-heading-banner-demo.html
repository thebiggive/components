Heading Banner Component Demo

This page demonstrates the usage of the new biggive-heading-banner component.

Basic Usage

The example above shows the basic usage of the heading banner component with a tall height, coral background for text, and custom focal point for the image.

With Logo and Slug

This example demonstrates the heading banner with additional optional properties:

  • logo: An object with URL and alt text for a logo image
  • slug: A smaller text that appears above the main title
  • height: Set to "short" for a more compact banner

As above but inside a biggive-page-section element

Normally the header would be outside of any section. Putting the header inside a section restricts its width.

Component Properties

Property Type Description Required
main-title string Main title text for the banner Yes
main-image-url string URL for the main banner image Yes
focal-point object Focal point for image positioning (x and y values as percentages 0-100) Yes
teaser string Teaser text that appears below the main title Yes
background-colour string Background color for the banner Yes
text-background-colour string Background color for the text content area Yes
text-colour string Text color for all text content Yes
height "short" | "tall" Height variant of the banner No (default: "tall")
logo object Optional logo object with URL and alt text No
slug string Optional slightly smaller text to appear above the main title No (default: "")

Notes

  • The component uses Shadow DOM for style encapsulation
  • Line breaks in the teaser text are automatically converted to <br/> elements
  • Hex color values can be provided with or without the # prefix
  • The focal point controls the positioning of the background image, useful for responsive layouts
